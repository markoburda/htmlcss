Quotations explained

The Block Quotation element

The HTML Element (or HTML Block Quotation Element) indicates that the enclosed text is an extended quotation. Usually, this is rendered visually by indentation. A URL for the source of the quotation may be given using the cite attribute, while a text representation of the source can be given using the cite element.

The Inline Quotation element

The HTML q element indicates that the enclosed text is a short inline quotation. Most modern browsers implement this by surrounding the text in quotation marks. This element is intended for short quotations that don't require paragraph breaks; for long quotations use the blockquote element.

Two things are infinite: the universe and human stupidity; and I'm not sure about the universe.

- Albert Einstein

I am so clever that sometimes I don't understand a single word of what I am saying.

- Oscar Wilde

Life is what happens to us while we are making other plans.

- Allen Saunders

The Citation element

The HTML Citation element is used to describe a reference to a cited creative work, and must include the title of that work, for example Fahrenheit 451 Novel by Ray Bradbury

The Inline Code element

The HTML code element displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code. By default, the content text is displayed using the user agent's default monospace font.

print("change da world my final message. Goodbye")

del and ins elements

The HTML element represents a range of text that has been added to a document. You can use the element to similarly represent a range of text that has been deleted from the document.

The Preformatted Text element

The HTML pre element represents preformatted text which is to be presented exactly as written in the HTML file. The text is typically rendered using a non-proportional ("monospace") font. Whitespace inside this element is displayed as written.

The Sample Output element

The HTML Sample Element is used to enclose inline text which represents sample (or quoted) output from a computer program. Its contents are typically rendered using the browser's default monospaced font (such as Courier or Lucida Console).

Please press '8' to question nothing

time element

The HTML time element represents a specific period in time. It may include the datetime attribute to translate dates into machine-readable format, allowing for better search engine results or custom features such as reminders.

The deadline of this project is at 00:00 on April 19, Saturday
